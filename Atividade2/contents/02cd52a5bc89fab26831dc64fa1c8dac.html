Acompanha serviço

UMLUseCase
Untitled :: Model :: Acompanha serviço

Description

Acompanha Serviço - Caso de Uso responsável por acompanhar o serviço, localização do Trabalhador no local de serviço (Cliente).

Fluxo Principal do Cliente:

1 - O Ator "Cliente" seleciona a opção para visualizar sua agenda de serviços marcados.

2 - O sistema informa os serviços agendados.

3 - O Cliente seleciona um serviço para visualizar seus detalhes.

4 - O sistema informa com mais detalhe o pedido. Como por exemplo: Tipo de serviço, Descrição do serviço, Valor do serviço, Dia e hora do serviço, Nome e endereço agendado.

Fluxo Principal do Trabalhador:

1 - O Ator "Trabalhador" seleciona a opção para visualizar sua agenda de serviços marcados.

2 - O sistema informa os serviços agendados.

3 - O Trabalhador seleciona um serviço para visualizar seus detalhes.

4 - O sistema informa com mais detalhe o pedido. Como por exemplo: Tipo de serviço, Descrição do serviço, Valor do serviço, Dia e hora do serviço, Nome e endereço do cliente.
