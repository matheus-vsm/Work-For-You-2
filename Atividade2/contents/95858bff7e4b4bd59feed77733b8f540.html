Avaliação

UMLUseCase
Untitled :: Model :: Avaliação

Description

Avaliação - Caso de Uso responsável pela avaliação, feita pelo Cliente, do serviço realizado pelo Trabalhador.

Fluxo Principal:

1 - Após o serviço for finalizado, o Cliente é direcionado para uma aba de avaliação do serviço.

2 - Cliente seleciona de 0 a 5 estrelas.

3 - Cliente pode ou não colocar alguma observação sobre o serviço prestado ou sobre o Trabalhador.

Fluxo Alternativo:

1 - Após o serviço for finalizado, o Cliente é direcionado para uma aba de avaliação do serviço.

2 - Cliente resolve pular a avaliação.
