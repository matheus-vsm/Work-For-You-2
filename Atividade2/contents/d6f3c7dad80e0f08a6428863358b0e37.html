Verifica se há serviços disponiveis

UMLUseCase
Untitled :: Model :: Verifica se há serviços disponiveis

Description

Verificar Se Há Serviços Disponíveis - Caso de Uso responsável pelo Trabalhador, por verificar se há algum serviço, para escolher a demanda que seja possível de ser realizado por ele.

Fluxo Principal:

1 - Após o Cliente Solicitar o Serviço, é feita uma verificação para ver se o serviço feito está disponível.

2 - Serviço está disponível e é enviado para a verificação do serviço por parte do Trabalhador.

Fluxo Alternativo:

1 - Após o Cliente Solicitar o Serviço, é feita uma verificação para ver se o serviço feito está disponível.

2 - Serviço não está disponível e é notificado para o Cliente para que seja feita outra solicitação.
