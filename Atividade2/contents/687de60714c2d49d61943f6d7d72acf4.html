Login

UMLUseCase
Untitled :: Model :: Login

Description

Login - Caso de Uso responsável por realizar o login no software.

Fluxo Principal:

1 - Ator coloca seus dados para realizar o Login.

2 - Ele clica no botão "entrar".

3 - Ator é direcionado ao menu

Fluxo Alternativo:

2 - Clica no botão "entrar".

3 - Mensagem de erro aparece ("E-mail ou senha inválidos")

4 - Ator é direcionado à tela de cadastro.
