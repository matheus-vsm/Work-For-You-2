Solicita serviço

UMLUseCase
Untitled :: Model :: Solicita serviço

Description

Solicita Serviço - Caso de Uso responsável por solicitar o serviço do Cliente.

Fluxo Principal:

1 - Cliente solicita no sistema o serviço desejado

2 - O sistema informa uma lista de Trabalhadores disponíveis junto com o valor do serviço.

3 - O Cliente seleciona o Trabalhador dentro da lista informada pelo sistema.

4 - O sistema informa um calendário para escolha da data e horário.

5 - O Cliente escolhe a data e horário desejada.

Fluxo Alternativo:

1º

1 - O Cliente solicita no sistema o serviço desejado.

2 - O sistema não consegue informar os Trabalhadores, pois não há nenhuma disponibilidade no momento.

3 - A solicitação é cancelada.

2º

2 - O sistema envia o pedido para o Trabalhador analisar se realizará ou não ou serviço.

3 - Caso o Trabalhador demore mais de dois dias para responder se realizará ou não o serviço solicitado, o Cliente receberá o reembolso do valor pago.
