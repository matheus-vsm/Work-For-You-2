Aprovação

UMLUseCase
Atividade 6 :: Diagrama de Casos de Uso :: Aprovação

Description

Aprovação - Caso de Uso responsável pela aprovação do pagamento pelo banco responsável usado pelo Cliente.

Fluxo Principal:

1 - Após o pagamento é enviado para aprovação.

2 - Adquirente aprova o pagamento.

3 - Pedido é enviado ao Trabalhador.

Fluxo Alternativo:

1 - Adquirente não aprova o pagamento.

2 - Cliente recebe a notificação de pagamento não aprovado.
