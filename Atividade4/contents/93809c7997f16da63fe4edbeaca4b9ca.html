Status Serviço

UMLUseCase
Atividade 4 :: Diagrama de Casos de Uso :: Status Serviço

Description

Aceita Serviço - Caso de Uso responsável pelo Trabalhador, aceitar o serviço solicitado pelo Cliente.

Fluxo Principal:

1 - Trabalhador é notificado de um novo pedido.

2 - Trabalhador se direciona à aba de visualização de novos pedidos solicitados.

3 - Trabalhador seleciona um pedido para ver mais detalhes.

4 - Trabalhador aceita o pedido.

5 - Pedido vai para a aba de pedidos em andamento.

Fluxo Alternativo:

1º

1 - Trabalhador se direciona à aba de visualização de novos pedidos solicitados.

2 - Não há novos pedidos.

2°

1 - Trabalhador seleciona um pedido para ver mais detalhes.

2 - Trabalhador nega o pedido.

3 - Trabalhador procura outro pedido.
