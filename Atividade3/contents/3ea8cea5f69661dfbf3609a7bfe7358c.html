Realiza Serviço

UMLUseCase
Atividade 3 :: Diagrama de Casos de Uso :: Realiza Serviço

Description

Realiza Serviço - Caso de Uso responsável pela realização do serviço solicitado pelo Cliente.

Fluxo Principal:

1 - Trabalhador notifica o Cliente que está indo ao local do trabalho.

2 - Realiza o serviço.

3 - Trabalhador e Cliente finalizam o pedido.
